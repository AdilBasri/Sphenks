## âœ¨ Key Features of the Build System

### Smart Resource Loading (`resource_path()`)
The game uses an intelligent path resolver that:
1. First checks PyInstaller's `_MEIPASS` folder
2. Then checks next to the executable
3. Falls back to development directory

This means the **same code works** whether you:
- Run from Python source
- Run the compiled EXE
- Run from any directory

### Multi-Format Audio Support
The game handles:
- `.wav` files
- `.mp3` files
- `.ogg` files

### Font Management
All 8 languages' fonts are bundled:
- English, Turkish, German, Spanish
- Chinese (custom font), Portuguese, French, Italian
